# arduino-security-system
very easy arduino sequrity system powere by arduino. this project uses some very simple materials. 
1. 1x Arduino Uno
2. 1x Analog Joystick
3. 1x I2C Lcd
4. 5x push buttons
5. 4x 330 ohm resistors
6. 1x piezo buzzer
7. multiple wires
8. 1x RGB led
9. 1x Servo motot
